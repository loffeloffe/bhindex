Proof-of-concept Metadata-system on top of BitHorde

BHIndex serves as a testing-framework and proof-of-concept system for BitHorde, enabling a generic database for asset-metadata. Data can be altered and synchronized between nodes. The database are schemaless, the views are based on a criteria-selection extension-system.

Movie data is gotten from IMDB, and:
  Information courtesy of
  The Internet Movie Database
  (http://www.imdb.com).
  Used with permission.

TV data is fetched from TheTVDB, using tvdb_api from https://github.com/dbr/tvdb_api
